## 六、如何扩展

### 添加更多签文

编辑 `miniprogram/data/signs.js`，在 SIGNS 数组中添加：

```javascript
{
  id: 'sign_007',
  number: 7,
  title: '上签',
  poem: '诗句一\n诗句二\n诗句三\n诗句四',
  meaning: '签义解释...',
  advice: '建议...',
  category: 'gji',
  tags: ['标签1', '标签2']
}
```

### 修改主题颜色

编辑 `miniprogram/app.wxss`，修改：
```scss
--primary-color: #D4AF37;  /* 改为其他颜色 */
```

### 修改应用名称

编辑 `miniprogram/app.json`：
```json
"navigationBarTitleText": "新名称"
```
